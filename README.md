8. This time besides the contests button you'll find a Matches button. Make any match you want.

9. scroll below and live the match.

10. Now same go to the Admin tab and trigger any event selecting a player.

11. trigger the events such that the players selected by user (you) perform well in the match to earn points. (points get updated in the leaderboard tab of admin and you can also view the points and get live notifications by logging in back with user account).

12. So basically what happens is if you luckily select players who perform well you get top of the leaderboard and finally get the price.

13. Now navigate back to the Admin tab and scroll below and end/finish the match. After that the top player gets rewarded. (You can login back to user and view notifications and view your wallet for the same).
